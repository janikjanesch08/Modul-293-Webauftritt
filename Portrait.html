Meine Projekte

HTML, CSS Webseite

Während meiner Tätigkeit im Betrieb habe ich eine Webseite mit HTML und CSS erstellt, die sich auf den Verkauf von Häusern spezialisierte. Die Webseite umfasste eine Startseite, auf der die wichtigsten Informationen präsentiert wurden, sowie eine Produktseite, die verschiedene Immobilienangebote darstellte. Zudem beinhaltete sie ein Kontaktformular, über das Interessenten einfach und schnell Anfragen stellen konnten.

Sprachgesteuerte KI

In den letzten 2,5 Monaten habe ich eine eigene KI mit Sprachsteuerung entwickelt. Diese KI akzeptiert ausschließlich mündliche Eingaben und gibt die Ergebnisse sowohl mündlich als auch in Textform aus. Durch diese innovative Technologie wird die Interaktion mit Computern und Geräten noch intuitiver und benutzerfreundlicher gestaltet.

Ämtliplan

Als ich im Betrieb das Thema Java durchgenommen habe, erstellte ich einen eigenen Ämtliplan. Dieser ermöglicht das Erfassen, Ausgeben, Erledigen und Löschen von Aufgaben. Mit meiner Java-Kenntnis konnte ich effektive Tools für die Organisation entwickeln.

Casino

Mit Java habe ich mein eigenes funktionierendes Casino entwickelt, das Roulette und einarmige Banditen bietet. Spieler können eigene Benutzer erstellen, Guthaben hinzufügen und die oben genannten Spiele genießen. Ein spannendes Unterfangen, das meine Programmierfähigkeiten herausgefordert hat!
